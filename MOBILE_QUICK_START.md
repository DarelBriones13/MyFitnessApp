# 📱 Mobile Quick Start

Get your workout app on your phone in 5 minutes!

---

## 🚀 3 Simple Steps

### Step 1: Generate Icons (1 minute)
1. Open `generate-icons.html` in your browser
2. Click "Download 192x192 Icon" → Save as `icon-192.png`
3. Click "Download 512x512 Icon" → Save as `icon-512.png`

### Step 2: Deploy (2 minutes)

**Option A: Netlify (Easiest)**
1. Go to [netlify.com](https://netlify.com)
2. Sign up (free)
3. Drag your project folder onto the page
4. Get URL: `https://your-app.netlify.app`

**Option B: GitHub Pages**
1. Create GitHub repository
2. Upload all files
3. Settings → Pages → Enable
4. Get URL: `https://username.github.io/repo-name`

**Option C: Local Network (No Internet)**
```bash
# On your computer
python -m http.server 8000 --bind 0.0.0.0

# Find your IP
ipconfig  # Windows
ifconfig  # Mac/Linux

# On your phone (same WiFi)
# Go to: http://YOUR_IP:8000
```

### Step 3: Install on Phone (1 minute)

**Android (Chrome):**
1. Open your app URL in Chrome
2. Tap menu (⋮) → "Add to Home Screen"
3. Tap "Add"

**iPhone (Safari):**
1. Open your app URL in Safari
2. Tap Share button → "Add to Home Screen"
3. Tap "Add"

## 🆘 Troubleshooting

### Can't Install?
- **Android**: Use Chrome (not Firefox)
- **iPhone**: Use Safari (not Chrome)
- Wait for page to fully load

### Doesn't Work Offline?
1. Open app while online
2. Visit all tabs once
3. Then try offline

### Videos Don't Play Offline?
- Normal! Videos need internet
- Thumbnails work offline
- All other features work offline
